### web技术——Comet——长轮询

#### 项目介绍

​	基于node的express框架实现的简易Comet——长轮询

#### 启动方式

​	进入项目所在文件夹，在node命令行下运行`node app.js`

#### 使用方式

​	多人可同时打开此网站，并在文本框中输入文字，服务器端会监听数据，并将内容广播到，所有链接此网站的控制台上，实现简易的聊天室。
